# Autômatos  Finito
Este repositório contém a implementação de um **analisador léxico**, desenvolvido como parte do TDE (*Trabalho Discente Efetivo*) da disciplina de Linguagens Formais.
O objetivo do projeto é realizar a análise léxica de sentenças simulando um autômato finito.


---


## Como rodar?
O projeto pode ser acessado via [***Github Pages***](https://luizfelipebuaszczyk.github.io/automato_finito/). Ou se preferir, o projeto pode ser facilmente acessado localmente, basta clonar o repositório =)


## Tecnologias
O projeto é simples, utiliza-se apenas tecnologias WEB da sua forma pura, sem nenhum *framework* ou biblioteca extra.
